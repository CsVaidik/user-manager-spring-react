## Setup & Run Locally  

### Prerequisites  
- Java 17+  
- Node.js 16+  
- MySQL 8.0  

### Backend (Spring Boot)  
```bash
cd backend  
./mvnw spring-boot:run  # Starts on http://localhost:8080  
```
### Frontend (React Js)
```bash
cd client  
npm install  
npm start  # Runs on http://localhost:3000  
```
